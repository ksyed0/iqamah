# Adhan Audio Files

Place MP3 or M4A audio files in this folder, then add them to the Xcode project target.

## Naming Convention
- adhaan_1.mp3  (Adhaan 1 — default)
- adhaan_2.mp3  (Adhaan 2)
- adhaan_3.mp3  (Adhaan 3)
- etc.

## Sources (public domain / permissive)
- mp3quran.net
- islamicfinder.org
- Your own recordings

## Adding to Xcode
1. Right-click the Media folder in Xcode navigator
2. "Add Files to iqamah..."
3. Ensure "Add to targets: iqamah" is checked
